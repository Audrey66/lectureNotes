How to prove two propositional logic statements are 
equivalent?

    Approach 1: two truth assignments
    for every possible truth assignment, the two statements have the same output

    Approach 2: 
        deduction with two proofs
        statement1 |- statement2
        statement2 |- statement1




How to prove propositional logic statements are 
NOT equivalent?

find one truth assignment where the two statements have different outputs

cant use deduction to prove inequivability


What is satisfiability? 

need one truth assignment where the statement outputs true

NOT satisfiability-
    full truth table to hwo the statement is always false

- Determining whether an argument is valid or invalid
    (what is an argument?)
    premises, conclustion

    valid: 
        if for every truth assignment where all premises are true, the conclusion is also true

    how prove valid?
        truth table with all premises and conclusion
        look at truth assignments that make all premises true,
        make sure they make conclusion true 

        deduction
    how prove invalid?
        find one truth assignment where all premises are true by conclusion false
        cant prove invalid with deduction
